- Responsive design for mobile and desktop

## Tech Stack

- React.js
- CSS
- JavaScript

## Installation

Clone the repo and run the app locally:

```bash
git clone https://github.com/Mohan20030731/Todo-List.git
cd Todo-List
npm install
npm start
```
